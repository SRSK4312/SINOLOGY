## Копирование на сервер
Подключится к серверу по ssh или другими методами,
установить Node, npm на сервер. 
Создать папку проекта, скопировать в неё git репозиторий при помощи команды git-clone

## Иницализация

```bash
$ npm install
```

## Запуск приложения

```bash
# production mode
$ npm run start:prod

# development
$ npm run start

# watch mode
$ npm run start:dev
```
